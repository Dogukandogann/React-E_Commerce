<h1>React Alışveriş Sitesi Projesi</h1>
<h2>Description</h2>
<p>Bu proje, React kullanarak geliştirilmiş bir alışveriş uygulamasıdır. Uygulama, çeşitli ürünleri gösterir, sepete ekleme ve çıkarma işlemleri yapılmasına izin verir ve toplam tutarı hesaplar. </br>
 </br>
<strong>Ana Sayfa:</strong> Swipper slayt + 4.5  rating üzeri popüler ürünlerin bulunduğu sayfa. </br>
<strong>Ürün Sayfası:</strong>  Tüm ürünlerin listelendiği sayfa. </br>
<strong>Sepet İşlemleri:</strong>  Ürünleri sepete ekleyebilir, çıkarabilir, miktarını artırabilir ve toplam tutarı görüntüleyebilirsiniz. </br>

<strong>Kullanılan Teknolojiler </strong> </br>
React.js </br>
Axios (Veri alışverişi için kullanıldı) </br>
useContext (Veri yönetimi için kullanıldı) </br>
Swiper (Ana menüde slayt gösterisi oluşturmak için kullanıldı)</p> </br>
